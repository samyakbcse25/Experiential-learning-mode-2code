What is the CSS Box Model?
How browsers calculate size and spacing for every HTML element.

Every HTML element is treated as a rectangular box by the browser. The CSS Box Model describes the four layers that make up that box, from the inside out:

  • content The actual text or image — its size is set with width and height.
  • padding Transparent space inside the border, between content and the edge.
  • border A visible line drawn around the padding area.
  • margin Transparent space outside the border — pushes other elements away.
💡

Pro tip: Always add box-sizing: border-box; — this includes padding and border inside the declared width, making layouts predictable.

Property Reference
Quick lookup for all Box Model CSS properties used above.
Property Example Value Description
background-color #818cf8 Fill colour of the element's content + padding area.
padding 24px 32px Inner spacing — top/bottom & left/right.
margin 0 0 16px 0 Outer spacing — top · right · bottom · left.
border 2px solid #6366f1 Width, style, and colour of the border line.
border-radius 10px Rounds each corner of the border box.
box-sizing border-box Includes padding & border inside the declared width.
width 100% Total width of the element (with border-box).
